## 项目介绍

本项目为 [Vant](https://youzan.github.io/vant/#/zh-CN/intro) 的 uni-app 移植版。目标是实现与 Vant 完全一致的 API，支持通过 uni-app 一次编译到多平台，目前主要支持 H5 和小程序端。
