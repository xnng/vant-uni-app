## 介绍

本项目为 [Vant](https://youzan.github.io/vant/#/zh-CN/intro) 的 uni-app 移植版。实现了与 Vant 完全一致的 API，支持通过 uni-app 一次编译到多平台。

## 安装

```
# 通过 npm 安装
npm i vant-uni-app

# 通过 yarn 安装
yarn add vant-uni-app
```
